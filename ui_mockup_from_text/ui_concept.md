# UI Concept - CodeVerse: Academic Pro

## Purpose
CodeVerse is a gamified programming learning platform that aligns with university curricula, allowing students to master coding through bite-sized lessons.

## Users
- **Students:** To learn programming and complete university-specific modules.
- **University Admins:** To create custom learning paths and monitor student progress.

## Core Screens
- **Learning Path (The Map):** A visual journey of lessons tailored to the user's university curriculum.
- **Practice Arena:** An interactive screen where students solve coding challenges to earn XP.
- **Leaderboard:** A competitive view showing student ranks within their university.

## Actions
- **Daily Lesson:** Complete a coding exercise to keep the streak alive.
- **Switch Curriculum:** Admins can update the teaching plan for the semester.
- **Claim Rewards:** Exchange earned points for virtual certificates.
