# UI Concept - CodeVerse: Academic Pro

## Purpose
CodeVerse is a gamified programming learning platform (similar to Duolingo) that aligns directly with university curricula. It allows students to master coding through bite-sized lessons while following their specific university’s syllabus.

## Users
- **Students:** To learn programming, compete in rankings, and complete university-specific modules.
- **University Admins/Teachers:** To create custom learning paths, upload teaching materials, and monitor student progress.
- **System Moderators:** To manage global leaderboards and platform health.

## Core Screens
- **Learning Path (The Map):** A visual journey of lessons (e.g., Python Basics, SQL Data structures) tailored to the user's university curriculum.
- **Practice Arena:** An interactive screen where students solve coding challenges to earn "XP" and maintain "Streaks."
- **Leaderboard/Rankings:** A competitive view showing student ranks within their university and globally.
- **Admin Dashboard:** A control panel for teachers to manage academic plans, see student performance analytics, and assign badges.
- **Profile & Inventory:** A place to view achievements, badges, and the "Rank" status (e.g., Bronze, Silver, Diamond).

## Actions
- **Daily Lesson:** Complete a coding exercise to keep the streak alive.
- **Switch Curriculum:** Admins can update the teaching plan based on the current semester.
- **Global Challenge:** Join weekly competitive coding matches against other universities.
- **Claim Rewards:** Exchange earned points for virtual items or certificates.
